# 鑄造 SPL 代幣

![鑄造 SPL 代幣挑戰](/graphics/challenge-banners/typescript-mint-an-spl-token.png)

SPL 代幣程式，特別是使用 TypeScript 鑄造代幣，應該是你作為 Solana 開發者旅程的第一步。

如果你對 SPL 代幣程式不熟悉，請先參考[這個課程](/zh-HK/courses/spl-token-with-web3js)！

在這個挑戰中，我們將實現四個簡單的指令：

- **建立鑄幣**：使用原始指令或 SDK 提供的抽象指令建立一個`Mint Account`。
- **初始化鑄幣**：使用原始指令或 SDK 提供的抽象指令初始化已建立的`Mint Account`。*鑄幣應設置為 6 位小數，鑄幣權限設置為運行代碼的錢包，且無凍結權限*。
- **建立關聯代幣帳戶**：使用原始指令或 SDK 提供的抽象指令建立並初始化一個`Associated Token Account`。
- **鑄造 2100 萬代幣**：將 2100 萬（21,000,000）新建立的代幣鑄造到新建立的`Associated Token Account`。

**注意**：如果你不熟悉如何使用 TypeScript 與 SPL-Token 程式，應該先閱讀 [SPL Token with Web3.js](/zh-HK/courses/spl-token-with-web3js) 以熟悉我們在這次挑戰中需要的 SDK。
