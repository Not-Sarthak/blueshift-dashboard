<ChallengeTitle>Pinocchio Escrow</ChallengeTitle>

Sekarang saatnya untuk menguji apa yang telah Anda pelajari. Dengan menggunakan apa yang telah Anda pelajari dari pelajaran ini, buatlah program escrow menggunakan framework Pinocchio yang memenuhi persyaratan berikut:

<RequirementList>
  <Requirement title="Tantangan 1: Membuat penawaran escrow">
    Program Anda harus memungkinkan pembuat untuk membuat penawaran escrow. Tentukan instruksi untuk melakukan ini menggunakan diskriminator instruksi dengan nilai `0`.
  </Requirement>

  <Requirement title="Tantangan 2: Mengambil penawaran escrow">
    Program Anda harus memungkinkan pengambil untuk mengambil penawaran escrow. Tentukan instruksi untuk melakukan ini menggunakan nilai diskriminator instruksi `1`.
  </Requirement>

  <Requirement title="Tantangan 3: Membatalkan penawaran escrow">
    Program Anda harus memungkinkan pembuat untuk membatalkan penawaran escrow dan mengembalikan dana yang di-escrow. Pastikan instruksi Anda didefinisikan dengan nilai diskriminator instruksi `2`.
  </Requirement>
</RequirementList>
