# Program Memo Pinocchio

Selamat datang di tantangan pertama Anda menggunakan Pinocchio! Tujuan Anda adalah membuat program memo sederhana yang mencatat pesan ke blockchain. Ini adalah pengenalan yang bagus tentang cara kerja instruksi di Pinocchio.

Program ini hanya memiliki satu instruksi dan tidak memiliki akun. Program ini langsung menggunakan `instruction_data` sebagai sumber string, kemudian mencatat string ini menggunakan syscall `sol_log`.

Program Anda juga harus memiliki Id program `22222222222222222222222222222222222222222222` untuk menyelesaikan tantangan ini.
