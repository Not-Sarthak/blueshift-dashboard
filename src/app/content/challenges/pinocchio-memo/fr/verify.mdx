# Programme Mémo Pinocchio

Bienvenue dans votre premier défi utilisant Pinocchio ! Votre objectif est de créer un programme mémo simple qui enregistre un message sur la blockchain. C'est une excellente introduction au fonctionnement des instructions dans Pinocchio.

Le programme n'a qu'une seule instruction et aucun compte. Il utilise directement le `instruction_data` comme source de la chaîne de caractères, puis enregistre simplement cette chaîne en utilisant l'appel système `sol_log`.

Votre programme devrait également avoir un identifiant de programme `22222222222222222222222222222222222222222222` pour réussir le défi.
