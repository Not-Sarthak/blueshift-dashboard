# Pinocchio Memo-Programm

Willkommen zu deiner ersten Herausforderung mit Pinocchio! Dein Ziel ist es, ein einfaches Memo-Programm zu erstellen, das eine Nachricht in der Blockchain protokolliert. Dies ist eine großartige Einführung in die Funktionsweise von Anweisungen in Pinocchio.

Das Programm hat nur eine Anweisung und keine Konten. Es verwendet direkt die `instruction_data` als Quelle für den String und protokolliert diesen String einfach mit dem `sol_log` Syscall.

Dein Programm sollte auch eine Programm-ID von `22222222222222222222222222222222222222222222` haben, um die Herausforderung zu bestehen.
