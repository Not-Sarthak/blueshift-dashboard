# Fazit

Du kannst jetzt dein Programm gegen unsere Unit-Tests testen und deine NFTs beanspruchen!

Beginne mit dem Erstellen deines Programms, indem du folgenden Befehl in deinem Terminal ausführst:

```
cargo build-sbf
```

Dies generiert eine `.so` Datei direkt in deinem `target/deploy` Ordner.

Klicke nun auf die Schaltfläche `take challenge` und lege die Datei dort ab!
