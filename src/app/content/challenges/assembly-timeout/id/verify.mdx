<ChallengeTitle>Timeout sBPF Assembly</ChallengeTitle>

Sekarang setelah Anda mulai merasa nyaman menulis Assembly, alih-alih menggunakan `sol_get_clock_sysvar` cobalah mencapai tujuan yang sama dengan memasukkan akun `Clock`.

Buat program yang hanya memiliki satu instruksi. Program tersebut harus:
- Menerima akun `Clock` (berukuran 40).
- Menerima slot u64 (8-byte) dalam data instruksi.
- Menggunakan 8-byte pertama dari akun `Clock` untuk mengambil slot saat ini.
- Memeriksa apakah slot saat ini lebih besar dari slot yang diberikan dalam instruksi. Jika ya, kembalikan kode error 1.

> Untuk mendapatkan offset data instruksi yang tepat, Anda dapat menggunakan [alat kami](https://sbpf.xyz). Ingat untuk menambahkan satu akun dan atur panjang datanya sebagai 40!

Mulailah dengan menginstal paket sBPF seperti yang dijelaskan di [bagian alat](/id/courses/introduction-to-assembly/tooling) dari kursus Pengantar Assembly dan bangun program Anda menggunakan perintah berikut di terminal Anda:

```
sbpf build
```

Ini akan menghasilkan file `.so` di folder `target/deploy` Anda.

Sekarang klik tombol `take challenge` dan letakkan file tersebut di sana!

<RequirementList>
  <Requirement title="Tantangan 1: Menguji Timeout">
    Program Anda harus berhasil ketika slot yang lebih tinggi dari slot saat ini diberikan.
  </Requirement>

  <Requirement title="Tantangan 2: Menguji Kegagalan Timeout">
    Program Anda harus gagal ketika slot yang lebih rendah dari slot saat ini diberikan.
  </Requirement>
</RequirementList>
