# Programme Memo Anchor

Bienvenue dans votre premier défi utilisant le framework Anchor sur Solana ! Votre objectif est de **créer un programme memo simple qui enregistre un message** sur la blockchain. C'est une excellente introduction au fonctionnement des instructions dans Anchor.

Le programme n'a qu'une seule instruction et prend en entrée :

- Une entrée `String`
- Aucun compte

Puis enregistre simplement cette chaîne de caractères en utilisant la macro Anchor `msg!()`.

Votre programme devrait également avoir un identifiant de programme `22222222222222222222222222222222222222222222` pour réussir le défi.
