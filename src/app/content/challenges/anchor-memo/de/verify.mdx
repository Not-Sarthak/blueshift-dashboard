# Anchor Memo-Programm

Willkommen zu deiner ersten Herausforderung mit dem Anchor-Framework auf Solana! Dein Ziel ist es, **ein einfaches Memo-Programm zu erstellen, das eine Nachricht** in der Blockchain protokolliert. Dies ist eine großartige Einführung in die Funktionsweise von Anweisungen in Anchor.

Das Programm hat nur eine Anweisung und nimmt Folgendes entgegen:

- Eine `String` Eingabe
- Keine Konten

Dann protokolliert es einfach diesen String mit dem Anchor `msg!()` Makro.

Dein Programm sollte auch eine Programm-ID von `22222222222222222222222222222222222222222222` haben, um die Herausforderung zu bestehen.
