# Program Memo Anchor

Selamat datang di tantangan pertama Anda menggunakan framework Anchor di Solana! Tujuan Anda adalah **membuat program memo sederhana yang mencatat pesan** ke blockchain. Ini adalah pengenalan yang bagus tentang cara kerja instruksi di Anchor.

Program ini hanya memiliki satu instruksi dan menerima:

- Input `String`
- Tidak ada akun

Kemudian cukup mencatat string ini menggunakan makro Anchor `msg!()`.

Program Anda juga harus memiliki Id program `22222222222222222222222222222222222222222222` untuk menyelesaikan tantangan.
