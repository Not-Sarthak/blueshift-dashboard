# 結論

恭喜你！你已完成「使用 LiteSVM 測試」課程。現在你已全面了解如何使用 LiteSVM，一個快速且輕量的測試框架，將 Solana 虛擬機直接引入你的測試流程，實現快速迭代和精確的程式測試。

<ArticleSection name="你所學到的內容" id="what-you-learned" level="h2" />

在這個課程中，你掌握了 LiteSVM 的核心概念：
- **性能優勢**：了解 LiteSVM 的內部虛擬機如何比傳統測試框架提供顯著更快的執行速度
- **帳戶管理**：創建和配置系統帳戶、程式帳戶以及 SPL 代幣帳戶，以進行全面的測試場景
- **交易執行**：設置、模擬和執行交易，並正確檢查結果
- **進階控制**：操作系統變數、控制運行時行為以及自定義測試環境

<ArticleSection name="下一步" id="next-steps" level="h2" />

現在你已具備使用 LiteSVM 為你的 Solana 程式構建快速且可靠測試的能力！以下是一些建議的下一步：
1. 應用到你的項目：將 LiteSVM 整合到你現有的 Solana 程式測試套件中，親身體驗性能提升
2. 探索進階測試：利用 LiteSVM 的帳戶操作功能，測試其他框架難以重現的複雜邊界情況和失敗場景
3. 性能基準測試：利用 LiteSVM 的速度，在開發和 CI/CD 流程中運行全面的測試套件
4. 加入社群：在 Discord 上與其他使用 LiteSVM 的開發者交流，分享測試策略和優化方法
5. 保持更新：關注 [LiteSVM 儲存庫](https://github.com/LiteSVM/litesvm)，了解最新功能、語言綁定和性能改進

> 特別感謝 [Aursen](https://x.com/exoaursen) 和 [Exotic Markets](https://x.com/exotic_markets_) 團隊創建並維護 LiteSVM，讓高性能的 Solana 程式測試對多語言開發者變得觸手可及。
