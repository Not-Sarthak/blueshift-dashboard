# Kesimpulan

Selamat! Anda telah menyelesaikan kursus `Create your SDK with Codama`. Anda sekarang memiliki dasar yang kuat dalam memahami cara kerja Codama IDL, mulai dari konsep intinya hingga detail implementasi praktisnya.

## Apa yang Telah Anda Pelajari

Sepanjang kursus ini, Anda telah mendapatkan pengetahuan penting tentang:
- Apa itu Codama dan bagaimana `nodes`, `visitors` dan `renderers` digunakan
- Cara membangun Codama IDL dari awal
- Cara membangun Codama IDL dari Anchor IDL dan memperbaruinya jika diperlukan

## Langkah Selanjutnya

Anda sekarang siap untuk mulai membangun SDK pertama Anda dengan Codama! Cara terbaik untuk memperkuat pengetahuan Anda adalah melalui praktik langsung. Kami mendorong Anda untuk:

1. Pergi ke [bagian Tantangan](/id/challenges)
2. Membuat SDK untuk salah satu program
3. Menggunakan SDK untuk membangun beberapa pengujian untuk program Anda
4. Bergabung dengan komunitas kami untuk berbagi kemajuan Anda dan mendapatkan bantuan
