# Kesimpulan

Selamat! Anda telah menyelesaikan kursus Program Token2022. Sekarang Anda memiliki pemahaman yang kuat tentang bagaimana Token2022 bekerja di Solana dan perannya.

<ArticleSection name="Apa yang telah Anda pelajari" id="what-you-learned" level="h2" />

Sepanjang kursus ini, Anda telah mendapatkan pengetahuan penting tentang:
- Perbedaan mendasar antara program Legacy dan program Token2022 yang baru
- Bagaimana Token Extension bekerja dan bagaimana mereka berinteraksi dengan akun `Mint` dan `Token`
- Struktur data dari setiap Token Extension
