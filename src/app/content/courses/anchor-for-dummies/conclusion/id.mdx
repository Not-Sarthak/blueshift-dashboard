# Kesimpulan

Selamat! Anda telah menyelesaikan kursus Pengantar Anchor. Sekarang Anda memiliki dasar yang kuat dalam memahami cara kerja Anchor, mulai dari konsep intinya hingga detail implementasi praktisnya.

## Apa yang Telah Anda Pelajari

Sepanjang kursus ini, Anda telah memperoleh pengetahuan penting tentang:
- Dasar-dasar framework Anchor
- Cara menyusun program Anchor
- Memahami diskriminator program
- Praktik terbaik untuk membangun program Solana yang aman

## Langkah Selanjutnya

Anda sekarang siap untuk mulai membangun program pertama Anda dengan Anchor! Cara terbaik untuk memperkuat pengetahuan Anda adalah melalui praktik langsung. Kami mendorong Anda untuk:

1. Pergi ke [bagian Tantangan](/id/challenges)
2. Mulai dengan latihan yang ramah untuk pemula
3. Bangun dan uji program Anchor pertama Anda
4. Bergabung dengan komunitas kami untuk berbagi kemajuan Anda dan mendapatkan bantuan

Ingat, setiap pengembang hebat memulai dengan program pertama mereka. Jangan takut untuk bereksperimen dan membuat kesalahan; begitulah cara kita belajar dan berkembang!
