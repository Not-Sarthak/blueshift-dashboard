# Kesimpulan

Selamat! Anda telah menyelesaikan kursus Testing dengan Surfpool. Sekarang Anda memiliki pemahaman komprehensif tentang cara menggunakan Surfpool, sebuah framework pengujian yang kuat yang membawa status mainnet langsung ke lingkungan pengembangan lokal Anda untuk pengujian Cross-Program Invocation yang mulus.

<ArticleSection name="What you've learned" id="what-you-learned" level="h2" />

Sepanjang kursus ini, Anda telah menguasai konsep-konsep penting Surfpool:
- **Simulasi mainnet**: Bagaimana Surfpool mengambil akun mainnet tepat waktu, menghilangkan setup kompleks untuk pengujian CPI
- **Kontrol lanjutan**: Melakukan perjalanan waktu ke epoch/slot/timestamp manapun, menghentikan produksi blok, dan memanipulasi variabel sistem
- **Modifikasi akun**: Mengatur parameter pasokan SOL, mentransfer akun antar program, dan memperbarui data akun token

<ArticleSection name="Next Steps" id="next-steps" level="h2" />

Anda sekarang dilengkapi untuk membangun pengujian komprehensif untuk program Solana yang kompleks menggunakan Surfpool! Berikut adalah beberapa langkah selanjutnya yang direkomendasikan:
1. Terapkan ke Proyek Anda: Integrasikan Surfnet ke dalam suite pengujian program Solana Anda yang sudah ada.
2. Bergabung dengan Komunitas: Terhubung dengan pengembang lain yang menggunakan Surfpool di Discord.
3. Tetap Update: Ikuti [repositori Surfpool](https://github.com/txtx/surfpool) untuk fitur terbaru dan peningkatan kinerja
