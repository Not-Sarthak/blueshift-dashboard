# Conclusion

Félicitations ! Vous avez terminé le cours sur les tests avec Surfpool. Vous avez maintenant une compréhension complète de l'utilisation de Surfpool, un framework de test puissant qui intègre l'état du réseau principal directement dans votre environnement de développement local pour des tests d'invocation inter-programmes (CPI) fluides.

<ArticleSection name="Ce que vous avez appris" id="what-you-learned" level="h2" />

Tout au long de ce cours, vous avez maîtrisé les concepts essentiels de Surfpool :
- **Simulation du réseau principal** : comment Surfpool récupère les comptes du réseau principal juste à temps, éliminant ainsi la configuration complexe pour les tests CPI
- **Contrôle avancé** : voyager dans le temps vers n'importe quelle époque/slot/horodatage, mettre en pause la production de blocs et manipuler les variables système
- **Modification de compte** : définir les paramètres d'approvisionnement en SOL, transférer des comptes entre programmes et mettre à jour les données des comptes de jetons
