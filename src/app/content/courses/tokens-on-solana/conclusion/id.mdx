# Kesimpulan

Selamat! Anda telah menyelesaikan kursus Token di Solana. Sekarang Anda memiliki pemahaman yang kuat tentang bagaimana token bekerja di Solana dan perannya.

<ArticleSection name="Apa yang telah Anda pelajari" id="what-you-learned" level="h2" />

Sepanjang kursus ini, Anda telah memperoleh pengetahuan penting tentang:
- Berbagai jenis token dan mengapa token tersebut penting.
- Bagaimana token bekerja di Solana dibandingkan dengan ekosistem lainnya.
- Apa itu akun Mint, Token, dan Associated Token.
- Tindakan apa saja yang dapat dilakukan pada token ini.

Anda sekarang siap untuk mulai menggunakan program token baru dan ekstensinya untuk memperkuat program Anda! Berikut adalah beberapa langkah selanjutnya yang direkomendasikan:

1. Pelajari lebih lanjut tentang cara menggunakan program ini dengan typescript di [bagian SPL-Token dengan Web3Js](/id/courses/spl-token-with-web3js)
2. Jelajahi [bagian Tantangan](/id/challenges) untuk latihan praktis
3. Bergabunglah dengan komunitas kami untuk berbagi ide, implementasi, dan mendapatkan bantuan
