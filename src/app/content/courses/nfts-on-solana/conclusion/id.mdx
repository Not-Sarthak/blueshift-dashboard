# Kesimpulan

Selamat! Anda telah menyelesaikan kursus komprehensif NFT di Solana dan mengembangkan pemahaman menyeluruh tentang token non-fungible dalam Solana.

<ArticleSection name="Apa yang telah Anda pelajari" id="what-you-learned" level="h2" />

Sepanjang kursus ini, Anda telah memperoleh pengetahuan penting tentang:
- Properti inti NFT: ketidakdivisibilitas, keunikan, pasokan tetap, pengelompokan, dan metadata
- Program Metaplex Token Metadata: `Metadata`, `Master Edition` dan akun `Token Record`
- Program Metaplex Core: Desain akun tunggal untuk efisiensi yang lebih baik dan sistem plugin mereka

<ArticleSection name="Langkah Selanjutnya" id="next-steps" level="h2" />

Anda sekarang siap untuk mulai membuat NFT pertama Anda! Berikut adalah beberapa langkah selanjutnya yang direkomendasikan:

1. Pelajari lebih lanjut tentang cara menggunakan program-program ini di dokumentasi resmi [Metaplex](https://developers.metaplex.com).
2. Jelajahi [bagian Tantangan](/id/challenges) untuk latihan praktis
3. Bergabunglah dengan komunitas kami untuk berbagi ide, implementasi, dan mendapatkan bantuan
