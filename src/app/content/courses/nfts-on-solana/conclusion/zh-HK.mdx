# 結論

恭喜你！你已完成全面的 Solana 上的 NFT 課程，並對 Solana 上的非同質化代幣有了深入的了解。

<ArticleSection name="你所學到的內容" id="what-you-learned" level="h2" />

在這個課程中，你已掌握以下重要知識：
- NFT 的核心屬性：不可分割性、唯一性、固定供應量、可分組性和元數據
- Metaplex Token Metadata 程式：`Metadata`、`Master Edition` 和 `Token Record` 帳戶
- Metaplex Core 程式：單一帳戶設計以提升效率及其插件系統
